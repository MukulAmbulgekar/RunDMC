The documents in this section configure tabbed-, or sidebar-based,
"features". To include features on a given page, use the
<ml:tabbed-features/> tag.

For example:

  <ml:tabbed-features>
    <ml:feature href="/features/mluc10.xml"/>
    <ml:feature href="/features/eclipse.xml"/>
    <ml:feature href="/features/office-toolkits.xml"/>
  </ml:tabbed-features>
